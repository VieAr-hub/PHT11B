1) Creating Directories / Folders
mkdir (directory_name)

2) Clear the terminal
clear

3) Deleting Directories / Folders
rmdir (directory_name)

4) Print / Present the Current Directoriy
pwd

5) List Command (show the content of the current directory)
ls

6) Change Directory
cd [directory_path]

7) Go back one directory
cd ..

8) Create a file
touch [file_name.ext]

9) Delete a file 
rm [file_name.ext]
